# Фотографии для карточек пончиков

Добавьте сюда изображения пончиков. Файлы будут использоваться в блоке «Наши пончики» (Products).

## Требуемые файлы

| Файл | Пончик |
|------|--------|
| `classic-glaze.webp` | Classic Glaze — Классическая глазурь |
| `choco-crunch.webp` | Choco Crunch — Шоколадный хруст |
| `berry-cloud.webp` | Berry Cloud — Ягодное облако |
| `salted-caramel.webp` | Salted Caramel — Солёная карамель |
| `matcha-dream.webp` | Matcha Dream — Мечта матча |
| `peanut-bomb.webp` | Peanut Bomb — Арахисовая бомба |

## Рекомендации

- **Формат:** WebP (или PNG/JPG — тогда нужно обновить пути в `site.config.ts`)
- **Размер:** примерно 400×400 px или больше (квадрат или близко к нему)
- **Фон:** нейтральный или прозрачный, чтобы пончик хорошо читался на карточке
